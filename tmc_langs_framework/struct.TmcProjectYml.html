A list of files or directories that will always be considered exercise files. extra_student_files takes precedence if a file is both an extra student file and an extra exercise file.

ยงforce_update: Vec<PathBuf>

A list of files that should always be overwritten by updates even if they are student files.

ยงtests_timeout_ms: Option<u32>

If set, tests are forcibly stopped after this duration.

ยงno_tests: Option<NoTests>

Marks the exercise as not containing any tests.
