Expand description

A builder for accessing an object property.

The builder methods make it convenient to get and set properties as well as to bind and call methods.

let x: f64 = obj
    .prop(&mut cx, "x")
    .get()?;

obj.prop(&mut cx, "y")
    .set(x)?;

let s: String = obj.method(&mut cx, "toString")?.call()?;
