Struct BindOptions

Source 
pub struct BindOptions<'a, 'cx: 'a> { /* private fields */ }
Expand description

A builder for making a JavaScript function call like parseInt("42").

The builder methods make it convenient to assemble the call from parts:

let x: f64 = parse_int
    .bind(&mut cx)
    .arg("42")?
    .call()?;
