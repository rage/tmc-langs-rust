Expand description

radium provides a series of helper traits providing a uniform API for interacting with both atomic types like AtomicUsize, and non-atomic types like Cell<T>.

This crate is #![no_std]-compatible, and uses no non-core types.

For details, see the documentation for Radium.

Additionally, radium exports type aliases that map to the atomic types in core::sync::atomic when they exist, and fall back to Cell wrappers when the atomic is missing. These are accessible through the types module; you can use these names for a guaranteed-portable symbol with best-effort atomic behavior.
